# Análise de Algoritmos de Ordenação
Atividade prática solicitada pelo Prof. Dr. André Luiz Satoshi Kawamoto na disciplina de Estrutura de Dados do PPGCC-CM (2026/1).

## Como executar

1. Clone o repositório.
2. Acesse o diretório ``/src``:
```bash
cd src
```
3. Compile:
```bash
make
```
4. Execute:
```bash
cd .. && ./bin/programa
```
5. Opcionalmente, limpe o diretório com o binário após executar:
```bash
make clean
```
<hr>

Adicionalmente, é possível compilar e executar utilizando apenas um comando:
```bash
make run
```
